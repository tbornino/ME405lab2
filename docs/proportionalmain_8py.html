Detailed Description

This code should be ron as main for a PROPORTIONAL only controller and was developed before the PID controller was implemented.

See main.py for the main code where the PID controller is implemented. The main code to run on the microcontroller to complete a step response on a motor. Includes functionality to set the setpoint and controller gain. The time and motor position are printed to the console as a two column list after the step response.
